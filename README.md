# Introduction Program

## Summary
For this project, I created a simple C++ program that introduces who I am. The program uses `cout` to print information about my name, life goals, current interests, dreams, and where I see myself in the next 1, 3, 5, and 7 years.

## How I Did It
I created a C++ file called `main.cpp`. Inside the program, I used the `iostream` library and `cout` statements to display my introduction. I also used the `int` type to store the numbers 1, 3, 5, and 7 for the future year goals.
